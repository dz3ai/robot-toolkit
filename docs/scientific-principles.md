# 科学原理

robot-toolkit 的算法实现基于以下机器人学和数学原理。

## 1. Denavit-Hartenberg 参数

### 问题

如何用最少的参数描述串联机械臂中相邻关节之间的几何关系？

### 原理

DH 参数法 (Denavit, Hartenberg, 1955) 用四个参数 `(a, α, d, θ)` 描述两个相邻旋转轴之间的空间关系：

```
从坐标系 {i-1} 到坐标系 {i} 的变换：

T = Rot_z(θ) · Trans_z(d) · Trans_x(a) · Rot_x(α)

    ┌ cos θ  -sin θ · cos α   sin θ · sin α   a · cos θ ┐
    ┌ sin θ   cos θ · cos α  -cos θ · sin α   a · sin θ ┐
    ┌  0       sin α           cos α            d         ┐
    └  0        0               0               1         ┘
```

- `θ` (theta): 绕 z 轴的旋转 — 旋转关节的变量
- `d`: 沿 z 轴的平移 — 移动关节的变量
- `a`: 沿 x 轴的平移 — 两轴之间的最短距离 (连杆长度)
- `α` (alpha): 绕 x 轴的旋转 — 两轴之间的夹角 (连杆扭角)

### 为什么用 DH 参数？

4 个参数恰好是最少自由度：空间中两个轴之间的关系由公垂线长度 `a` 和夹角 `α` 确定，加上沿轴的偏移 `d` 和初始旋转 `θ`。相比一般的 4×4 齐次变换 (6 个自由度)，DH 参数自动消除了冗余。

## 2. 正向运动学 (Forward Kinematics)

### 问题

已知各关节角度，求末端执行器的位姿 (位置 + 姿态)。

### 原理

正向运动学是 DH 参数的直接应用：将所有连杆变换矩阵连乘：

```
T_0→n = T_01(q₁) · T_12(q₂) · ... · T_{n-1,n}(qₙ)
```

这是一个确定性的映射：关节空间 q ∈ ℝ⁶ → 任务空间 T ∈ SE(3)。

### 数值特性

变换矩阵的乘法保持 SO(3) 群结构（旋转矩阵始终正交），但由于浮点累积误差，多次连乘后可能轻微偏离正交。在对精度敏感的应用中，可以对结果重新正交化。

## 3. 逆运动学 (Inverse Kinematics)

### 问题

已知目标末端位姿，求各关节角度。

### 难点

- 非线性映射：FK 是关节角的非线性函数
- 多解性：同一末端位姿可能对应多组关节角
- 奇异位形：某些构型下 Jacobian 降秩，丧失自由度
- 解可能不存在：目标在工作空间之外

### 采用的方法：阻尼最小二乘法 (Damped Least Squares, DLS)

也称为 Levenberg-Marquardt 方法。迭代步骤：

```
Δq = Jᵀ · (J · Jᵀ + λ² · I)⁻¹ · e
```

其中：
- `J` 是 6×6 几何雅可比矩阵
- `e` 是 6 维误差向量 (位置误差 3 + 姿态误差 3)
- `λ` 是阻尼因子

**阻尼因子的作用**：

当 Jacobian 接近奇异 (`cond(J·Jᵀ)` 很大) 时，`(J·Jᵀ)⁻¹` 中的某些分量会趋于无穷。阻尼项 `λ²·I` 限制这些分量的大小，代价是在奇异方向上降低求解精度。

**自适应阻尼**：

```
λ = λ₀ · (1 + 0.1 · log₁₀(cond(J·Jᵀ)))
```

条件数越大（越接近奇异），阻尼越强。这确保算法在奇异附近仍然稳定收敛。

**姿态误差计算**：

使用轴角表示法从旋转误差矩阵 `R_err = R_target · R_currentᵀ` 提取：

```
θ = arccos((tr(R_err) - 1) / 2)
axis = [R₃₂-R₂₃, R₁₃-R₃₁, R₂₁-R₁₂] / (2·sin θ)
orient_error = θ · axis
```

## 4. 几何雅可比矩阵 (Geometric Jacobian)

### 定义

雅可比矩阵建立了关节速度空间与末端速度空间之间的线性映射：

```
ẋ = J(q) · q̇
```

其中 `ẋ = [v; ω]` 是末端的线速度和角速度。

### 推导

对于旋转关节 i，其对末端线速度和角速度的贡献为：

```
J_v,i = z_i × (p_ee - p_i)    # 叉积给出切线方向的速度
J_ω,i = z_i                    # 角速度就是关节轴方向
```

这是从正向运动学解析求导得到的，与数值有限差分相比精确且高效。

### 奇异值分解与可操作度

雅可比矩阵的奇异值反映了机器人在当前构型下的运动能力：

```
w = √(det(J · Jᵀ))    # 可操作度 (manipulability)
```

可操作度趋近零意味着接近奇异位形。

## 5. 递推牛顿-欧拉算法 (RNEA)

### 问题

已知各关节的位置、速度、加速度，计算各关节所需的驱动力矩。

### 运动方程

机器人的动力学方程为：

```
τ = M(q)·q̈ + C(q, q̇)·q̇ + G(q)
```

- `M(q)`: 关节空间惯性矩阵 (正定、对称)
- `C(q, q̇)·q̇`: 科里奥利力和离心力项
- `G(q)`: 重力项

### 算法原理

RNEA 基于牛顿-欧拉方程，通过两次递推求解：

**正向递推** (基座 → 末端)：传播运动学

```
对每个连杆 i:
  ω_i = ω_{i-1} + q̇_i · z_i         # 角速度
  α_i = α_{i-1} + q̈_i · z_i + ω_{i-1} × (q̇_i · z_i)  # 角加速度
  a_origin_i = a_{i-1} + α_i × r_i + ω_i × (ω_i × r_i)  # 关节原点线加速度
  a_com_i = a_{i-1} + α_i × r_com + ω_i × (ω_i × r_com) # 质心线加速度
```

**反向递推** (末端 → 基座)：传播力/力矩

```
对每个连杆 i (从 n 到 1):
  F_i = m_i · a_com_i                 # 牛顿第二定律
  N_i = I_i · α_i + ω_i × (I_i · ω_i) # 欧拉方程
  f_i = f_{i+1} + F_i                 # 力平衡
  n_i = n_{i+1} + r_com × F_i + N_i   # 力矩平衡
  τ_i = n_i · z_i                     # 投影到关节轴
```

### 复杂度

O(n) — 线性于关节数。对于 6 自由度机器人，每步仅 6 次循环。

### 重力处理技巧

基座加速度初始化为 `-g`（负重力向量），而非零。这样在后续递推中，重力效应自然包含在加速度传播中，无需单独计算。

## 6. 正向动力学与惯性矩阵

### 惯性矩阵

使用有限差分法从逆动力学提取：

```
H[i,j] = ∂τᵢ/∂q̈ⱼ ≈ (τ(q̇=0, q̈=ε·eⱼ) - τ(q̇=0, q̈=0)) / ε
```

这是 Composite Rigid Body Algorithm (CRBA) 的近似版本。精确的 CRBA 通过递推计算，但有限差分法对于 6 自由度系统足够精确且实现简单。

### 正向动力学

给定关节力矩 τ，求解加速度：

```
q̈ = H(q)⁻¹ · (τ - C(q,q̇)·q̇ - G(q))
```

通过将逆动力学与惯性矩阵组合实现。

## 7. URDF 运动链解析

### URDF 与 DH 参数的对应

URDF 使用任意齐次变换描述关节关系，而 DH 参数要求特定的参数化。转换过程：

1. 从关节原点和轴向提取偏移 `d`（沿轴向分量）和臂长 `a`（垂直于轴向分量）
2. 连杆扭角 `α` 通过相邻关节轴的夹角确定
3. 质心从 URDF 坐标系变换到 DH 坐标系

这是一个近似转换——精确的转换需要考虑关节轴的精确几何关系。对于大多数工业机器人（关节轴平行或正交），近似足够精确。
